# 🕸️ Advanced Web Scraper: Quotes to Scrape

This is an advanced Python scraper for [quotes.toscrape.com](https://quotes.toscrape.com). It collects all quotes across all pages and saves them as either `.csv` or `.json`.

## Features
- 🔄 Scrapes **all pages**
- 🧾 Save to **CSV or JSON**
- 📁 Organized in `output/` folder
- 📅 Timestamped filenames
- 🧠 Easy to modify for other sites

## How to Run

```bash
pip install -r requirements.txt
python scraper.py
